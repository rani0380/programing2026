01

Python 기본

Python은 문법이 비교적 간단하고 라이브러리가 풍부해 초보자가 접근하기 쉬운 프로그래밍 언어입니다. 코드 작성과 실행 흐름을 묻는 문제가 자주 출제됩니다.

대화형 셸

명령을 한 줄씩 입력하고 결과를 바로 확인할 수 있습니다. 간단한 계산, 문자열 출력, 변수 확인 문제와 연결됩니다.

>>> 5 + 3
8
>>> a = 5
>>> b = 4
>>> a + b
9
02

주석과 들여쓰기

주석은 프로그램 실행과 관계없는 설명입니다. 한 줄 주석은 `#`을 사용하고, 여러 줄 설명은 따옴표를 활용할 수 있습니다.

print("안녕하세요")  # 화면에 문장을 출력

'''
여러 줄 설명을 적을 때
사용할 수 있습니다.
'''
시험 포인트

Python은 중괄호 대신 들여쓰기로 코드 블록을 구분합니다. 조건문과 반복문에서 들여쓰기가 틀리면 오류가 발생합니다.

03

print와 input

`print()`는 값을 화면에 출력하고, `input()`은 사용자가 입력한 값을 문자열로 받습니다. 숫자로 계산하려면 `int()` 또는 `float()` 변환이 필요합니다.

name = input("이름을 입력하세요: ")
print(name, "님 안녕하세요")

a = int(input("숫자를 입력하세요: "))
print(a + a)
  • `sep` 옵션은 출력값 사이의 구분 문자를 정합니다.
  • `end` 옵션은 출력 후 마지막 문자를 정합니다.
  • 쉼표를 사용하면 여러 값을 한 번에 출력할 수 있습니다.
04

자료형

자료형은 데이터의 종류입니다. 변수에 저장되는 값에 따라 정수, 실수, 문자열, 참과 거짓 등이 구분됩니다.

자료형예시핵심
정수형10, -3소수점이 없는 숫자
실수형3.14소수점이 있는 숫자
문자열"hello"문자의 나열
논리형True, False참 또는 거짓
 
a = 10
b = 5
c = "프로그래밍"
print(type(a))
print(type(c))
05

자료형 변환

자료형 변환은 데이터를 다른 자료형으로 바꾸는 것입니다. 입력값은 기본적으로 문자열이므로 계산 전에 숫자형 변환이 필요합니다.

str(10)       # "10"
int("10")     # 10
float("12.5") # 12.5
list("hi")    # ["h", "i"]
자주 나오는 실수

`input()`으로 받은 값은 문자열입니다. `input()` 값끼리 더하면 숫자 덧셈이 아니라 문자열 연결이 됩니다.

06

연산자

연산자는 값을 계산하거나 비교할 때 사용하는 기호입니다. 산술 연산자와 비교 연산자는 실행 결과를 묻는 문제에 자주 등장합니다.

연산자의미예시
+더하기a + b
-빼기a - b
*곱하기a * b
/나누기a / b
//몫a // b
%나머지a % b
**거듭제곱a ** b
 
a = 10
b = 5
print(a + b)  # 15
print(a / b)  # 2.0
print(a ** b) # 100000
07

연산자 우선순위

여러 연산자가 한 식에 함께 나오면 정해진 우선순위에 따라 계산됩니다. 괄호가 있으면 괄호 안의 식이 가장 먼저 계산됩니다.

우선순위연산자의미
1**거듭제곱
2*, /, %, //곱하기, 나누기, 나머지, 몫
3+, -더하기, 빼기
4<, >, <=, >=, ==, !=관계 연산
5not논리 부정
6and논리곱
7or논리합
 
print(2 + 5 > 2 + 5 and not 3 * 3 > 10)
# False and not False
# False and True
# 결과: False
08

문자열 슬라이싱

문자열은 문자들이 순서대로 나열된 시퀀스 자료형입니다. 인덱스는 왼쪽에서 0부터 시작하고, 오른쪽에서 접근할 때는 -1부터 시작합니다.

text = "handaman nice"
print(text[0])      # h
print(text[5:8])    # man
print(text[8:])     # nice
print(text[::-1])   # ecin namadnah
  • `text[a:b]`는 a번 위치부터 b번 바로 앞까지 가져옵니다.
  • `text[:b]`는 처음부터 b번 바로 앞까지 가져옵니다.
  • `text[a:]`는 a번 위치부터 끝까지 가져옵니다.
  • `text[::-1]`은 문자열을 거꾸로 뒤집는 대표 패턴입니다.
09

문자열 함수

문자열 함수는 실행 결과를 묻는 문제가 자주 나옵니다. 함수 이름과 반환값의 의미를 짝으로 기억하세요.

함수기능예시 결과
format()문자열 형식 지정"ABC 123"
replace()문자열 바꾸기"ABC KBC"
split()기준 문자로 나누기리스트 반환
join()문자열 사이에 구분자 넣기"a/b/c"
count()문자 개수 세기1, 2, 0
strip()양쪽 공백 제거"abcde"
find()위치 찾기, 없으면 -10, 4, -1
index()위치 찾기, 없으면 오류오류 발생 가능
upper(), lower()대문자, 소문자 변환"ABC", "abc"
zfill()앞쪽을 0으로 채우기"0001"
 
text = "abc ABC"
print(text.replace("A", "K"))  # abc KBC
print(text.count("A"))         # 1
print("-".join("abc"))         # a-b-c

군집 자료형

군집 자료형은 여러 값을 하나의 이름으로 관리하는 자료형입니다. 리스트와 딕셔너리는 기출에서 실행 결과를 묻는 형태로 자주 출제됩니다.

자료형표현특징
리스트[1, 2, 3]순서가 있고 값 변경 가능
딕셔너리{"kor": 80}키와 값으로 저장
튜플(1, 2, 3)순서가 있고 값 변경 불가
세트{1, 2, 3}중복을 허용하지 않음
 
list1 = [1, 2, 3]
list2 = [4, 5]
list1.insert(2, list2)
print(list1)  # [1, 2, [4, 5], 3]

score = {"kor": 80, "eng": 90}
score["sci"] = 92
print(score)
리스트 함수

`append`, `insert`, `remove`, `sort`, `reverse`, `count`의 실행 결과를 구분하세요. `insert(위치, 값)`은 지정한 위치에 값을 끼워 넣습니다.
